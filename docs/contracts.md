## Mint Order

OpenSea Drops / SeaDrop quantity mints assign ids by the deterministic affine
order:

```text
tokenId = (mintOrdinal * 7321 + 7900) % 10000
```

`7321` is coprime with `10000`, so this order covers every token id exactly
once. The first mainnet mint assigned token id `7900`. Public minters cannot
choose exact token ids through the mint flow.

## Admin Disclosure

Important owner-controlled surfaces are documented in
[admin.md](admin.md). In short:

- `setRenderer()` can swap art until `lockRenderer()` is called.
- `lockRenderer()` freezes only the renderer/art path.
- Feeding cooldown, ETH withdrawal, royalty settings, SeaDrop configuration,
  transfer-validator configuration, ownership transfer/renounce, and Lifebuoy
  rescue flags are separate administrative surfaces.
